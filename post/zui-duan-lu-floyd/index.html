最短路——Floyd

二. 算法

  • 1. 思想：

    用二维数组dpdpdp记录当前iii点到jjj点的最短路。从不允许中转，再依次增加，可以从节点111到节点kkk进行中转，直到所有节点都可以中转。

  • 2. 实现：

    枚举起点iii，终点jjj和中转点kkk,将当前的最短路与iii到kkk再到jjj的最短路比较，更新。过程可看作dpdpdp。注意kkk是外层循环，表示可以用111到kkk号节点进行中转，并且要排除重点

  • 4. 代码
#include <cstdio>
long long dp[1005][1005], n, m;
void Floyd(){
	for (int k = 1; k <= n; k++) 
		for (int i = 1; i <= n; i++) 
			for (int j = 1 ; j <= n; j++) 
				if (i != k && j != i && j != k && dp[i][j] > dp[i][k] + dp[k][j]) 
					dp[i][j] = dp[i][k] + dp[k][j];
	return ;
}
int main() {
	scanf("%d", &n);
	for (int i = 1; i <= n; i++)
		for (int j = 1; j <= n; j++)
			scanf("%d", &dp[i][j]);
	Floyd();
	for (int i = 1; i <= n; i++){
		for (int j = 1 ;j <= n; j++) printf("%d ", dp[i][j]); 
		printf("\n");
	}
	return 0;
}
