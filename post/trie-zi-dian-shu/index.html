Trie字典树

2020-03-09 1 min read # 结

一. 定义：

用于处理字符串前缀问题

二. 思想

    1. 特点：利用字符串的公共前缀来减少查询时间。
    1. 实现：
    • I. 建树：将所有字符串合并成一棵树（用数组存，可以将字符转为下标，便于查找），字符转为节点，前缀相同的部分共用枝干，字符串结束用数组记录。
void _insert() {
    int len = strlen(s), root = 0;
    for(int i = 0; i < len; i++) {
        int id = s[i] - 'a';
        if(!trie[root][id]) trie[root][id] = ++tot;
        root = trie[root][id];
    }
    cnt[root]++;
}
  • II. 查询：查询有多少个子符串是你查询的字符串sss的前缀。顺着sss从trietrietrie的根一直往下 ， 记录你走过了多少个字符串的末尾 ， 直到无法继续向下走。
int _find() {
    int ans = 0, len = strlen(s), root = 0;
    for(int i = 0; i < len; i++) {
        int id = s[i] - 'a';
        root = trie[root][id];
        if(!root) return ans;
        ans += cnt[root];
    }
    return ans;
}
