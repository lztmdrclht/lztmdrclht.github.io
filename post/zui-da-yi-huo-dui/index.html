最大异或对

2020-03-09 2 min read # Trie

题目

校内OJ

分析

1.考点

Trie

2.思路

从二进制下手，思考异或本质，我们会发现位置越高越重要，所以我们要优先考虑高位。挨个比对显然不行，这时候，就可以用字典树了。

用trietrietrie存储所有数的二进制，就可以从最高位（根）开始查找，保证高位优先。而且trietrietrie中有且仅有所有可用的数，能保证查找到当前能查的是最优方案。只需枚举每个数，从trietrietrie中顺着匹配的数走，尽量找不同，最后取最大值就可以了。

代码

#include<iostream>
#include<cstdio>
#include<cmath>
using namespace std;
const int MAXN = 100005;
int n, tot = 2, trie[MAXN * 32][3], a[MAXN];
void _insert(int x) {
	int root = 1;
    for(int i = 30; i >= 0; --i) {
    	int u = x >> i & 1;
    	if(!trie[root][u]) trie[root][u] = tot++;
        root = trie[root][u];
	}
}
int _find(int x) {
	int root = 1, ans = 0;
    for(int i = 30; i >= 0; i--){
        int u = x >> i & 1;
        if(trie[root][u^1]){
            root = trie[root][u^1];
            ans += (1 << i);
        }
        else root = trie[root][u];
    }
    return ans;
}
int main() {
    scanf("%d", &n);
    for(int i = 1; i <= n; i++) {
        scanf("%d", &a[i]); 
		_insert(a[i]);
    }
    int ans = 0;
    for(int i = 1; i <= n; ++i) ans = max(ans, _find(a[i]));
    printf("%d\n", ans);
    return 0;
}
