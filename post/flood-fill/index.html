题目

校内OJ
洛谷

分析

1.考点

区间dp

2.思路

先看数据范围：n<5000n<5000n<5000 , O(n2n^2n2)?

目测区间dp，状态转移方程dp[i][j]=min(dp[i][j],dp[i][k]+dp[k][j]+1)dp[i][j]=min(dp[i][j],dp[i][k]+dp[k][j]+1)dp[i][j]=min(dp[i][j],dp[i][k]+dp[k][j]+1) ———O(n3n^3n3)，原地爆炸。

那么考虑优化，kkk是否必要。若iii与i+1i+1i+1颜色相同，则将两个视为同一连通块。我们的外层循环枚举长度，可以保证从短到长，也可以理解为将中央染成两侧的颜色进行扩张。既然能保证相邻的颜色不同，中间的连通块的颜色必然为两侧的，所以肯定与扩张的端点颜色不同。那么状态转移方程就推出来了：若两端颜色相同，dp[i][j]=dp[i+1][j−1]+1dp[i][j] = dp[i + 1][j - 1] + 1dp[i][j]=dp[i+1][j−1]+1,否则dp[i][j]=min(dp[i+1][j],dp[i][j−1])+1dp[i][j] = min(dp[i + 1][j], dp[i][j - 1]) + 1dp[i][j]=min(dp[i+1][j],dp[i][j−1])+1。外层循环长度，内层循环左端点，O(n^2)，愉快AC。(注意颜色在int范围内，需要离散化)

代码

#include <cstdio>
#include <cstring>
#include <iostream>
using namespace std;
const int MAXN = 5005;
int n, a[MAXN], dp[MAXN][MAXN], tot;
int main () {
    scanf ("%d", &n);
    for (int i = 1, u; i <= n; i++) {
        scanf ("%d", &u);
        if (u != a[tot]) a[++tot] = u;
    }
    for (int len = 1; len <= tot; len++)
        for (int i = 1; i + len <= tot; i++) {
            int j = i + len;
            if (a[i] == a[j]) dp[i][j] = dp[i + 1][j - 1] + 1;
            else dp[i][j] = min(dp[i + 1][j], dp[i][j - 1]) + 1;
        }
    printf ("%d\n", dp[1][tot]);
    return 0;
}
