三角形牧场

2020-03-09 2 min read # dp——背包

题目

校内OJ
洛谷

分析

1.考点

动态规划（背包）

2.思路

让我们先看看暴力吧，800* 800* 800（每条边最多长边长的一半），妥妥的MLE。那么想想怎么减小内存。

题中要求用到所有的木板，那么只需要记录三角形的两条边。哟西，省了一维。

定义状态bool dp[i][j]表示共用，能否拼成其中两边分别长i，j的三角形。外层循环k从1到n（前k个木板搭成第三边），第二层i(一边长)从1到tot（木板总长）/2（c<a+b），最内层j同i。记得判断三边长。

代码

#include <cstdio>
#include <iostream>
#include <cmath>
using namespace std; 
const int MAXN = 805;
using namespace std;
int n, a[55], sum;
double ans = -1;
bool dp[MAXN][MAXN];
bool check(int x, int y, int z)  {
    return x + y > z && x + z > y && y + z > x;
}
double hailun(double x, double y, double z){
    double p = (x + y + z) / 2;
    return sqrt(p * (p - x) * (p - y) * (p - z));
}
int main() {
    scanf("%d", &n);
    for(int i = 1; i <= n; i++) {
		scanf("%d", &a[i]); 
		sum += a[i];
	}
    dp[0][0] = 1;
    for(int k = 1; k <= n; k++) {
    	for(int i = sum / 2; i >= 0; i--) {
        	for(int j = sum / 2; j >= 0; j--) {
        	    if(((i - a[k] >= 0 && dp[i - a[k]][j]) || (j - a[k] >= 0 && dp[i][j - a[k]])) && check(i, j, sum - i - j)) {
			  	    dp[i][j] = true;
			  	    ans = max(ans, hailun(i, j, sum - i - j));
				} 
			}
		}
	}	
    if(ans != -1) printf("%lld\n", (long long)(ans * 100));
    else printf("-1\n");
    return 0;
}
