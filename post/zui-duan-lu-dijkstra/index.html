最短路——Dijkstra

发布于 2020-03-09 | 分类于 结 | 3分钟 | 507字数

一. 适用范围

I.单源最短路

II.无负权边

二. 算法

  • 1. 思想：

    设G=(V,E)是一个带权有向图，把图中顶点集合V分成两组。

    一组为已求出最短路径的顶点集合SSS,另一组为未确定最短路径的顶点集合UUU.每个顶点对应原点到此节点当前最短距离。

    初始时SSS中只有一个源点，以后每求得一条最短路径, 就将加入到集合SSS中，直到全部顶点都加入到SSS中。

    按最短路径长度的递增次序依次把第二组的顶点加入S中。因为已经是当前最短边，所以不可能通过其他点中转更新。【这也是为什么不能有负权边的原因】，然后更新UUU。

  • 2. 实现：

    用boolboolbool表示当前节点属于SSS或UUU。初始时SSS中只有原点，再从UUU中选取一个距离最小的顶点kkk加入SSS,以kkk为的中间点，更新U中各顶点的距离。重复以上操作直至所有节点都在SSS中。

#include<cstdio>
#include<iostream>
#include<cstring>
const int  MAXN = 10000, INF = 0x3f3f3f3f;
using namespace std;
bool vis[MAXN];
long long e[MAXN][MAXN], dis[MAXN], n, m, s, t, u, v, w;
void init() {
	for(int i = 0; i <= n; i++) {
		for(int j = 0; j <= n; j++) {
			if(i == j) e[i][j] = 0;
			else e[i][j] = INF;
		}
	}
}
void Dijkstra() {
	for(int i = 0; i <= n; i++) {
		vis[i] = 0;
		dis[i] = INF;
	}
	dis[s] = 0;
	int minn, cur;
	for(int k = 0; k < n; k++) {
		minn = INF;
		cur=0;
		for(int i = 0; i <= n; i++) {
			if(minn > dis[i] && !vis[i]) {
				minn = dis[i];
				cur = i;
			}
		}
		if(!cur) continue;
		vis[cur] = 1;
		for(int i = 0; i <= n; i++)
			if(e[cur][i])
				dis[i] = min(dis[i],dis[cur]+e[cur][i]);
	}
}
int main() {
	scanf("%d %d %d %d", &n, &m, &s, &t);
	init();
	for(int i = 0; i < m; i++) {
		scanf("%d %d %d", &u, &v, &w);
		e[u][v] = w;
		e[v][u] = w;
	}
	Dijkstra();
	printf("%d\n", dis[t]);
	return 0;
}
