「一本通 5.2 例 1」二叉苹果树

2020-03-09 2 min read # dp——树形dp

题目

校内OJ
洛谷

分析

1.考点

树形dp，背包

2.思路

这是一道板子题。

状态定义：dp[i][j]根为i的子树保留j根树枝。答案为dp[1][m]dp[1][m]dp[1][m](1为根结点，m为可保留树枝数)

顺序：从根向下搜索，每棵子树容量最大为其节点数，。每棵子树从一条边到所有边，挨个添加（背包）

代码

#include <iostream>
#include <cstdio>
#include <cstring>
#include <vector>
using namespace std;
const int MAXN = 105;
struct node {
    int v, w;
    node(){};
    node(int V, int W) {
        v = V;
        w = W;
    };
};
vector<node> G[MAXN];
int dp[MAXN][MAXN], n, m;
int dfs(int u, int fa) {
    int V = 1;
    for (int i = 0, v, w; i < G[u].size(); i++) {
        v = G[u][i].v;
        w = G[u][i].w;
        if (v == fa) continue;
        V += dfs(v, u);
        for (int j = V; j; j--)
            for (int k = 0; k < j; k++) 
            	dp[u][j] = max(dp[u][j], dp[u][j - k - 1] + dp[v][k] + w);
    }
    return V;
}
int main() {
    scanf("%d %d", &n, &m);
    for (int i = 1, u, v, w; i < n; i++) {
        scanf("%d %d %d", &u, &v, &w);
        G[u].push_back(node(v, w));
        G[v].push_back(node(u, w));
    }
    dfs(1, 0);
    printf("%d\n", dp[1][m]);
}
