校内OJ
洛谷

分析

1.考点

数论

2.思路

大水题， 枚举所有情况分析就好了，so easy。

先分析max， 因为胜一场333分，所有优先保证胜利，再是平局。

1.若s>=ns>=ns>=n，就可以保证每场比赛至少进一颗球，将所有输球安排在一场比赛，就能赢&n-1&场，再将剩余的胜球与&t&比较，决定最后一场胜负。

2.若s<ns<ns<n,则尽量将sss分散至尽量多的比赛，同样将将所有输球安排在一场比赛，保证胜利，剩下的"0:0"平局记得加分，同样最后根据ttt的大小决定最后一场是否平局。

再分析min,同样尽量输，但平局1，21，21，2场比胜一局得分少，需考虑

1.若t>=st >= st>=s，则比较尽量尽量多输，只赢一场，和一场不胜，至少平局。

2.若t<st<st<s,则尽量将ttt分散至尽量多的比赛，将s放在一场，记录平局场数。不要忘了将ttt与nnn比较，防止总场数超过nnn

代码

#include <cstdio>
#include <iostream>
using namespace std;
long long s, t, n,_max, _min;
int main() {
    while(~scanf("%lld %lld %lld", &s, &t, &n)){
        _max=0, _min = 0x3f3f3f;
        if(s >= n){
            _max = (n - 1) * 3;
            if(s - n + 1 > t) _max += 3;
            else if(s - n + 1 == t) _max += 1;
        }
        else {
            _max = 3 * s - 1;
            if(t == 0) _max++;
            _max += n - s;
        }
        if(t >= s){
            long long u = 3, v = 0;
            if(t < n - 1) u += n - 1 - t;
            if(n > t - s) v = n - (t - s);
            _min = min(u, v);
        }
        else {
            if(t >= n - 1) _min = 3;
            else _min = 3 + n - 1 - t;
        }
        printf("%lld %lld\n", _max, _min);
    }
    return 0;
}
